# Startup OS - Premium Frontend V1

Built with:
- **Next.js 14+ (App Router)**
- **Tailwind CSS v4**
- **Framer Motion** (Animations & 3D Tilt)
- **Lucide React** (Icons)

## Features
- **Premium Landing Page**: Glassmorphism, 3D Tilt Mockup, Scroll Animations.
- **Onboarding Flow**: Multi-step wizard with "Startup DNA" capture.
- **Dashboard**: Founder-view with "Health Score", Goals, and Team Pulse widgets.
- **Design System**: "Cyber Purple" theme, glass panels, beautiful gradients.

## Getting Started
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:3000`

## Structure
- `src/app/page.tsx`: Landing Page
- `src/app/onboarding/page.tsx`: Setup Flow
- `src/app/dashboard/page.tsx`: Main App Interface
- `src/components/ui/`: Reusable components (Button, Card, Input)
- `src/app/globals.css`: Tailwind v4 Theme & Utilities
